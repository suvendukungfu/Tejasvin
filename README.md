# Tejasvin - Hidden Heritage Repository

Welcome to the **Tejasvin** repository, home of the **Hidden Heritage** project. Hidden Heritage is a cognitive, spatial computing platform designed to treat history not as a static archive, but as a living signal.

## 📂 Repository Structure

This repository contains multiple components and iterations of the project:

- **[`hidden-heritage-mvp/`](./hidden-heritage-mvp/)** 🌟 _(Active Development)_
  The primary Minimum Viable Product (MVP) of the Hidden Heritage platform. It includes:
  - A React + Vite frontend with a custom Neural UX design.
  - A Node.js + Express backend.
  - Dual-database layer (MySQL & MongoDB).
  - WebXR spatial computing for 3D artifact viewing.
  - For full details, see the [MVP README](./hidden-heritage-mvp/README.md).

- **`Hidden-Heritage-Unvisited-Places1.0/`**
  An earlier version or specific module of the unvisited places explorer.

- **`frontend/`**
  Standalone frontend exploration or legacy prototype code.

## 🚀 Getting Started
